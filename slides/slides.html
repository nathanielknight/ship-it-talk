Starting point for packaging/installation tasks

Setuptools

📦 3rd party package (standard-ish)

Nice interface to distutils

greeter/
├── .git/...          
├── greeter
│   ├── ... (much code)
│   └── __init__.py
├── README.md
├── LICENSE
└── setup.py

Perform commands by running setup.py, eg: 


$ python setup.py install

setuptools.py


import setuptools

setuptools.setup(
    name="greeter",
    packages=["greeter"],
    version="0.0.0",
)
